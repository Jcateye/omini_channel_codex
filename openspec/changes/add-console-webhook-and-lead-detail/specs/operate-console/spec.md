## ADDED Requirements
### Requirement: View webhook delivery list
The system SHALL display CRM webhook delivery records with filter and pagination controls.

#### Scenario: Filter webhook deliveries
- **WHEN** an operator filters deliveries by status or event type
- **THEN** the system lists the matching records with status and timestamps

### Requirement: View lead detail
The system SHALL display a lead detail view with metadata, tags, and recent signals or rule matches.

#### Scenario: Open lead detail
- **WHEN** an operator selects a lead from the list
- **THEN** the system shows the lead detail view with its metadata and recent activity
