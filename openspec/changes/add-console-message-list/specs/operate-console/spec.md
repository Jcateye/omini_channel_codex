## ADDED Requirements
### Requirement: Console Message List
The console SHALL display a list of recent messages with status and provider message id.

#### Scenario: View recent messages
- **WHEN** a user opens the console message list panel
- **THEN** the most recent messages are displayed with status and external id.

### Requirement: Message List Filters
The console SHALL allow filtering the message list by status or channel.

#### Scenario: Filter messages
- **WHEN** a user filters by status or channel
- **THEN** the list only shows matching messages.
